static void detection_delay_timeout_handler ( void *  p_context)
static

Function for handling the timeout that delays reporting buttons as pushed.

The detection_delay_timeout_handler(...) is a call-back issued from the app_timer module. It is called with the p_context parameter. The p_context parameter is provided to the app_timer module when a timer is started, using the call app_timer_start. On app_timer_start the p_context will be holding the currently pressed buttons.

Parameters
[in]p_contextPointer used for passing information app_start_timer() was called. In the app_button module the p_context holds information on pressed buttons.
