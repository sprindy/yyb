Release notes

nRF51822 Beacon Version 1.1.1

Built on nRF51 SDK v7.1.0

Toolchains and devices used for testing and verification

  • ARM: MDK-ARM Version 5.12
  • s110_nrf51822_7.1.0_softdevice.hex
  • nRF51 SDK v7.1.0
  • PCA20006 v1.1.0 Beacon HW
  • PCA10000 v2.2.0 Dongle HW
  • Master Control Panel v3.7.1
  • nRFgo Studio v1.17.0

Changes

Bug fixes

  • A bug in SoftDevice <= 7.1.0 caused the bootloader to consume more current. This fix forces low power mode after sd_softdevice_disable() in bootloader.c @ line:315.
