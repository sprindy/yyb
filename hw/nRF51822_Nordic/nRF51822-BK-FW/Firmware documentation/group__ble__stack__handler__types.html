Detailed Description

This file contains the declarations of types required for BLE stack support. These types will be defined when the preprocessor define BLE_STACK_SUPPORT_REQD is defined.

Macro Definition Documentation

#define BLE_STACK_EVT_MSG_BUF_SIZE   0

Since the BLE stack support is not required, this is equated to 0, so that the softdevice_handler.h can compute the internal event buffer size without having to care for BLE events.
